# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.0.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this tool, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

### How to Report

1. Email: **niteshghimire0147@gmail.com**
2. Subject: `[SECURITY] WebVulnScanner - <brief description>`
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

### Response Timeline

- **Acknowledgment:** Within 48 hours
- **Initial Assessment:** Within 7 days
- **Fix/Patch:** Within 30 days for critical issues

### Scope

This policy covers:
- The scanner tool itself (`main.py`, `modules/`, `core/`, `utils/`, `reporter/`)
- CI/CD configurations

This policy does **not** cover:
- Target applications being scanned
- Third-party dependencies (report upstream)

## Responsible Use

This tool is designed for **authorized security testing only**. Using it against systems without explicit written permission is illegal. See [DISCLAIMER.md](DISCLAIMER.md).
